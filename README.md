# Educational-Data-PCA-Analysis

This project applies Principal Component Analysis (PCA) to an educational dataset containing detailed information on colleges and universities across the United States. The goal is to perform dimensionality reduction to uncover underlying patterns and relationships between institutional characteristics such as admissions, faculty, expenses, and graduation rates.
